### About
This repo is my attempt at a Gotdot 4 3D platformer course though Udemy, it covered basic examples of,
- Character movement
- Applying animation
- Camera movement
- Grid maps
- Moving platforms
- Moving and hostile objects
- Collectables
- Level advancement
- Menus
- Audio settings
- Display settings

With some basics down I look forward to future projects :)
